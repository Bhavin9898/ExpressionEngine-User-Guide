File Module
===========

The File Module allows you to display data associated with the files in
your `File Upload
Directories <../../cp/content/files/file_upload_preferences.html>`_.

-  :doc:`File Entries Tag <file_tag>`

In addition, the File module supports `File
Pagination <../channel/pagination_page.html>`_.
