Template Partials
=================

.. rst-class:: cp-path

**Control Panel Location:** :menuselection:`Developer Tools --> Template Partials`

.. Overview

In this section of the Control Panel you can create and edit
:doc:`Template Partials </cp/design/template_partials>`.

.. Permissions

.. Each Action

.. include:: /cp/design/_header.rst

Create New
~~~~~~~~~~

This will take you to the :doc:`create template form <form>`.

Partial Links
~~~~~~~~~~~~~

This will take you to the template partial's :doc:`edit form <form>`.

Manage
~~~~~~

The icons in the manage column perform actions on the template partial in its row.

Edit
^^^^

This will take you to the template partial's :doc:`edit form <form>`.

Find
^^^^

This will perform a search for the template partial, showing all the templates
that use it.

Bulk Actions
~~~~~~~~~~~~

The checkbox in the right-most column of the table selects a button for a bulk
action. When at least one checkbox is checked the bulk action dropdown menu and
submit button will be made available in the lower righthand corner of the table.

Remove
^^^^^^

The selected template partials will be removed. Removing a template partial
will cause a confirmation modal to appear that will summarize the action.

Export Partials
^^^^^^^^^^^^^^^

The selected template partials will be zipped and downloaded.
