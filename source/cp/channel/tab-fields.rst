Channel Fields Tab
======================

.. rst-class:: cp-path

**Control Panel Location:** :menuselection:`Developer --> Channels --> New/Edit Channel --> Fields`

.. Overview

This tab allows you to create and assign fields and field groups to your channel.

.. Screenshot (optional)

.. Permissions

Permission Restrictions
-----------------------

* Access settings: Design & Content
* Channels: Create Channel Fields
* Channels: Edit Channel Fields
* Channels: Delete Channel Fields
