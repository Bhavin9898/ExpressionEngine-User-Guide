Create/Edit Category Groups
===========================

.. rst-class:: cp-path

**Control Panel Location:** :menuselection:`Developer --> Categories --> New/Edit Category Group`

.. Overview

The category group form is broken out into 3 tabs: Details, Permissions, Fields.

.. Each Field


Details tab
***********

The :doc:`Details tab <tab-details>` holds fields for the category group name (used only for display in the control panel), the HTML formatting options, and whether the category group is available in Files, Channels or both.


Permissions tab
***************

The :doc:`Permissions tab <tab-permissions>` controls which member groups have permission to edit or delete categories.  The member group must have permission to edit categories in the :doc:`member group settings </cp/members/groups/form>` in order to be granted permission to edit a particular category group.


Fields tab
**********

The :doc:`Fields tab <tab-fields>` allow you to manage all custom fields for the category group, including deleting and creating new fields.
